기본 업데이트#

  1. 기본적으로 ubuntu를 업데이트 한다.

    sudo apt-get install
sudo apt-get update

zsh 설치#

  1. 먼저 zsh를 설치합니다.

    sudo apt install zsh

  2. 기본 셸로 설정

    sudo chsh -s $(which zsh)

  3. 실행 시 zsh를 실행되도록 지정한다.

    sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

Meslo Nerd Font 설치#

  1. 해당 폰트를 사용하지 않으면 아이콘 표시가 되지 않는다. 아래 링크를 통해 MesloLGS NF 폰트를 설정한다.

    romkatv/powerlevel10k

  2. WSL2에서 폰트를 변경합니다. 상단에서 마우스 우클릭 후 속성을 누른 뒤 폰트를 변경합니다.

powerlever10k 설치#

  1. zsh를 설치하면 oh-my-zsh라는 폴더가 생깁니다. 해당 폴더에 아래의 테마를 설치한다.

    git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k

  2. 설치 후 편집기를 이용하여 테마를 변경한다.

    vim ~/.zshrc

  3. 기존 테마인 ZSH_THEME=“robbyrussell"를 ZSH_THEME=“powerlevel10k/powerlevel10k” 로 변경한다.(i를 누르면 편집이 가능하며, 수정이 완료된 뒤 ESC를 누르고 :wq 입력)

    ZSH_THEME="powerlevel10k/powerlevel10k"

  4. 적용을 하셨다면 해당 창을 종료 후 다시 시작하면 아래와 같이 나오게 된다.

    (만약 나오지 않은 경우 명령어 source ~/.zshrc 입력)

  5. 여기서 본인의 스타일에 맞게 ()안의 답을 선택하면 된다. 저의 경우 패턴 선택 후 (2, 1, 4, 2, 1 1 1 2 1 12 2 1n1y)

powerlevel10k 재설정#

기존 설정을 바꾸고 싶다면 아래 명령어를 입력하면 다시 설정가능합니다.

p10k configure

폴더 테마 변경#

  1. 해당 테마를 변경하기 위해서는 ruby를 설치하여 colorls를 설치 해야 한다.

    sudo apt-get install ruby-full

  2. colorls 설치(만약 우분투 20.04 이상 버전인 경우 sudo apt install gcc make 실행 후 입력)

    sudo gem install colorls

  3. 설치가 완료됐으면 ~/.zshrc 가장 하단에 아래 코드를 추가한다.

    source $(dirname $(gem which colorls))/tab_complete.sh
alias ls=colorls

에러#

  1. 해당 에러가 발생하면

    bash: ll: command not found

  2. 이렇게 명령어를 입력하면 해결됩니다.

    echo "alias ll='ls -lGaf'" >> ~/.bash_profile

source ~/.bash_profile

터미널 색상 변경#

기본적으로 터미널의 색을 바꾸고 싶다면

위와 같이 설정에 들어가서 defaults 제이슨 값에

"defaults": {
  // color theme
  "colorScheme": "Campbell"
},

해당 테마이름을 지정하고, profiles제이슨 항목 안에

"schemes": [
  {
    "name": "Campbell",
    "cursorColor": "#FFFFFF",
    "selectionBackground": "#FFFFFF",
    "background": "#0C0C0C",
    "foreground": "#CCCCCC",
    "black": "#0C0C0C",
    "blue": "#4b6ed6",
    "cyan": "#3A96DD",
    "green": "#C50F1F",
    "purple": "#881798",
    "red": "#C50F1F",
    "white": "#CCCCCC",
    "yellow": "#C19C00",
    "brightBlack": "#767676",
    "brightBlue": "#3B78FF",
    "brightCyan": "#61D6D6",
    "brightGreen": "#16C60C",
    "brightPurple": "#B4009E",
    "brightRed": "#E74856",
    "brightWhite": "#F2F2F2",
    "brightYellow": "#F9F1A5"
  }
],

이렇게 지정해주면 된다. 공식 사이트(https://docs.microsoft.com/ko-kr/windows/terminal/customize-settings/color-schemes) 에서도 참조하면 된다.

위와 같은 방법으로 다른 테마를 적용하고 싶다면, https://windowsterminalthemes.dev/ 에서 테마를 정보를 위와 같이 넣어주면 된다.

글꼴 변경#

해당 글꼴을 넣어줘야 이미지를 볼 수 있다.


"defaults": {
	"fontFace" : "MesloLGS NF",
  "fontSize" : 13
},

탐색기에 WSL 연결하기#

  1. 탐색기 주소창에 아래의 주소값을 입력한다.
\\wsl$
  1. Unbutu 폴더에서 우측마우스 클릭 후 네트워크 드라이브 연결을 선택한다.
  2. 연결 후 원하는 이름으로 변경하면 아래와 같이 WSL 폴더내부를 볼 수 있다.
